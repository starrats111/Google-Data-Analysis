# 验证服务启动

## 检查服务状态

执行以下命令：

```bash
# 1. 检查服务进程
ps aux | grep uvicorn | grep -v grep

# 2. 测试健康检查
curl -s http://127.0.0.1:8000/health

# 3. 查看最新日志
tail -n 30 ~/Google-Data-Analysis/backend/run.log
```

## 如果服务没有运行

手动启动：

```bash
cd ~/Google-Data-Analysis/backend
source venv/bin/activate
nohup uvicorn app.main:app --host 0.0.0.0 --port 8000 > run.log 2>&1 &
sleep 5
curl -s http://127.0.0.1:8000/health && echo "✓ 服务运行正常" || tail -n 50 run.log
```

## 新功能已可用

### API端点自动检测
- 自动尝试多个可能的Rewardoo API端点
- 支持一键启用自动检测
- 检测成功后自动保存配置

### 使用方法
1. 打开同步数据对话框
2. 输入API Token
3. 点击"测试连接"
4. 如果出现404错误，点击"启用自动检测端点"按钮
5. 系统会自动找到正确的端点
