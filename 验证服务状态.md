# 验证服务状态

## 1. 检查服务是否正在运行
```bash
ps aux | grep uvicorn | grep -v grep
```

## 2. 测试健康检查端点
```bash
curl -s http://127.0.0.1:8000/health
```

## 3. 测试API端点（从外部）
```bash
curl -s https://api.google-data-analysis.top/health
```

## 4. 查看服务日志（实时）
```bash
tail -f ~/Google-Data-Analysis/backend/run.log
```

## 如果服务正常，应该看到：
- 进程在运行
- 健康检查返回成功
- 日志显示正常请求

## 如果需要重启服务：
```bash
# 停止服务
pkill -f "uvicorn app.main:app"
sleep 2

# 重新启动
cd ~/Google-Data-Analysis/backend
source venv/bin/activate
nohup uvicorn app.main:app --host 0.0.0.0 --port 8000 > run.log 2>&1 &
sleep 5

# 验证
curl -s http://127.0.0.1:8000/health && echo "✓ 服务运行正常" || echo "✗ 服务启动失败"
```
