MerchantDetails API
Summary
You can use this API to check merchant details status and acquire campaign links to monetize your traffic.
Request Information
URL:
https://admin.rewardoo.com/api.php?mod=medium&op=merchant_details
Return Format:
JSON
HTTP Request:
POST application/x-www-form-urlencoded
Required Post Parameters
