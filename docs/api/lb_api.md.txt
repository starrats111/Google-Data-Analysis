Transaction API
Summary
You can use this API to pull detailed transactions, commission status and many more.

Request Information
URL
https://www.linkbux.com/api.php?mod=medium&op=transaction_v2
Return Format
JSON
HTTP Request
GET
Required Parameters
token
Select Channel
begin_date
Transaction period. Format: YYYY-MM-DD (Data queries are only supported for dates starting from 2023/1/1.). Either transaction period or validation period is mandatory.
end_date
Transaction period. Format: YYYY-MM-DD
validation_date_begin
Validation period. Format: YYYY-MM-DD (Data queries are only supported for dates starting from 2023/1/1.). Either transaction period or validation period is mandatory.
validation_date_end
Validation period. Format: YYYY-MM-DD
