# 员工获取刷新令牌 - 在线工具版（最简单！）

## 🎉 好消息！

**现在不需要安装Python，也不需要访问本地文件！**

你可以直接在浏览器中使用在线工具获取刷新令牌。

---

## 🚀 超简单3步

### 第1步：打开在线工具

在浏览器中访问：
```
https://google-data-analysis.top/oauth-tool
```

或者：
```
https://api.google-data-analysis.top/oauth-tool
```

---

### 第2步：填写信息并授权

1. **如果系统已自动填充**客户端ID和密钥，直接点击"生成授权URL"
2. **如果没有自动填充**，向经理获取：
   - 客户端ID
   - 客户端密钥
3. 点击"生成授权URL"按钮
4. 在新窗口中打开授权页面
5. 使用你的Google账号完成授权
6. **复制显示的授权码**（一串很长的字符）

---

### 第3步：获取刷新令牌

1. 回到工具页面
2. 将授权码粘贴到输入框中
3. 点击"获取刷新令牌"
4. **复制显示的刷新令牌**

---

## ✅ 完成！

现在你有了刷新令牌，可以在系统中添加MCC账号了：

1. 登录系统
2. 进入"MCC账号"页面
3. 点击"添加MCC账号"
4. 填写：
   - MCC账号ID
   - 刷新令牌（刚才复制的）
5. 保存并测试连接

---

## ❓ 常见问题

### Q1: 访问工具页面显示404或无法打开

**解决方法：**
- 确认网址是否正确
- 联系经理确认系统是否正常运行
- 尝试使用不同的网址（.top 或 .pages.dev）

### Q2: 授权后没有显示授权码

**解决方法：**
- 确认是否完成了所有授权步骤
- 检查浏览器是否阻止了弹窗
- 尝试使用不同的浏览器

### Q3: 获取刷新令牌时提示错误

**解决方法：**
- 确认授权码是否正确复制（没有多余空格）
- 确认客户端ID和密钥是否正确
- 如果还是不行，联系经理

### Q4: 授权时提示"应用未验证"

**解决方法：**
1. 这是正常的（因为应用还在测试阶段）
2. 点击"高级"
3. 点击"继续前往 xxx（不安全）"
4. 继续授权
