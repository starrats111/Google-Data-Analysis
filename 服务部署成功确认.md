## 现在可以使用

### 对于LB、PM、BSH、CF平台：

1. **配置API URL**（必填）：
   - 打开"编辑账号"对话框
   - 找到对应的"API URL"字段
   - 输入正确的API基础URL

2. **配置API Token**：
   - 在同步对话框中输入Token
   - 或在账号备注中添加JSON配置

3. **同步数据**：
   - 选择日期范围
   - 点击"开始同步"

### 账号备注配置示例

```json
{
  "lb_api_url": "https://api.lb.com/api",
  "lb_token": "your_token"
}
```
