# BloomRoots - Lifestyle Blog Website

A beautiful, modern lifestyle blog website featuring six main categories: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

- **Modern Design**: Aurora gradient effects, clean minimalist layout
- **Six Categories**: Comprehensive coverage of lifestyle topics
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Article Pages**: Detailed article views with recommended products
- **Search Functionality**: Search articles by title, content, or category
- **Pagination**: Easy navigation through article listings
- **Smooth Animations**: Elegant transitions and hover effects

## File Structure

```
BloomRoots/
├── index.html          # Home page
├── category.html       # Category listing page
├── article.html        # Individual article page
├── styles.css          # Main stylesheet
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Usage

Simply open `index.html` in a web browser to view the website. All pages are static HTML files that work without a server.

### Navigation

- **Home**: View all articles on the homepage
- **Categories**: Click on any category in the navigation menu to filter articles
- **Articles**: Click on any article title to read the full article
- **Search**: Use the search box in the navigation bar to find articles

## Design Features

- **Aurora Gradient Background**: Beautiful color transitions throughout the site
- **Transparent Elements**: Glass-morphism effects on navigation and cards
- **Modern Typography**: Clean, thin sans-serif fonts
- **Smooth Animations**: Hover effects and page transitions
- **Product Recommendations**: Each article includes recommended products

## Categories

1. **Fashion & Accessories** - Latest fashion trends and style tips
2. **Health & Beauty** - Skincare, wellness, and beauty advice
3. **Home & Garden** - Interior design and gardening inspiration
4. **Travel & Accommodation** - Travel guides and destination recommendations
5. **Finance & Insurance** - Financial planning and investment strategies
6. **Food & Beverage** - Recipes, cooking tips, and culinary experiences

## Browser Support

This website works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Customization

To customize the website:

1. **Articles**: Edit the `articlesData` array in `script.js`
2. **Colors**: Modify CSS variables in `styles.css` (root section)
3. **Content**: Update HTML files directly
4. **Styling**: Adjust styles in `styles.css`
