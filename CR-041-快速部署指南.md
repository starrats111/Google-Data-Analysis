# CR-041 快速部署指南

## 📋 一键部署脚本

在服务器上执行以下命令，将以下内容保存为 `deploy_cr041.sh`：

```bash
#!/bin/bash
set -e

echo "=========================================="
echo "CR-041 部署和验证流程"
echo "=========================================="

# 1. 拉取最新代码
echo ""
echo "📥 步骤1：拉取最新代码..."
cd ~/Google-Data-Analysis
git pull origin main

# 2. 进入后端目录
echo ""
echo "📂 步骤2：进入后端目录..."
cd backend

# 3. 激活虚拟环境
echo ""
echo "🐍 步骤3：激活虚拟环境..."
source venv/bin/activate

# 4. 停止旧服务
echo ""
echo "🛑 步骤4：停止旧服务..."
pkill -f 'uvicorn.*app.main' || true
sleep 2

# 5. 启动新服务
echo ""
echo "🚀 步骤5：启动新服务..."
nohup uvicorn app.main:app --host 0.0.0.0 --port 8000 > /tmp/backend.log 2>&1 &
sleep 3

# 6. 验证服务健康状态
echo ""
echo "🏥 步骤6：验证服务健康状态..."
curl -s http://127.0.0.1:8000/health

# 7. 运行诊断脚本
echo ""
echo "🔍 步骤7：运行 L7D 数据准确性诊断..."
python scripts/diagnose_l7d_data_accuracy.py

# 8. 运行数据准确性验证
echo ""
echo "✅ 步骤8：运行数据准确性验证..."
python scripts/verify_data_accuracy.py

# 9. 查看服务日志
echo ""
echo "📋 步骤9：检查服务日志（最后30行）..."
tail -30 /tmp/backend.log

echo ""
echo "=========================================="
echo "✅ 部署和验证流程完成！"
echo "=========================================="
```

### 执行步骤：

1. **在服务器上创建脚本文件**
```bash
cat > ~/deploy_cr041.sh << 'EOF'
# 将上面的脚本内容粘贴到这里
EOF
```

2. **赋予执行权限**
```bash
chmod +x ~/deploy_cr041.sh
```

3. **执行部署脚本**
```bash
bash ~/deploy_cr041.sh
```

## 🐛 故障排查

### 问题1：服务启动失败
```bash
# 查看详细日志
tail -100 /tmp/backend.log

# 检查端口是否被占用
lsof -i :8000

# 强制杀死占用端口的进程
fuser -k 8000/tcp
```

### 问题2：诊断脚本报错
```bash

cd ~/Google-Data-Analysis/backend
python -c "from app.database import SessionLocal; db = SessionLocal(); print('✅ 数据库连接成功')"

python -c "from app.models.user import User; print('✅ 模型导入成功')"
```

### 问题3：数据不准确
```bash
# 查看 Google Ads 数据
cd ~/Google-Data-Analysis/backend
python << 'EOF'
from app.database import SessionLocal
from app.models.google_ads_api_data import GoogleAdsApiData
from datetime import date, timedelta

db = SessionLocal()
end_date = date.today() - timedelta(days=1)
begin_date = end_date - timedelta(days=6)

data = db.query(GoogleAdsApiData).filter(
    GoogleAdsApiData.date >= begin_date,
    GoogleAdsApiData.date <= end_date
).all()

print(f'总记录数: {len(data)}')
for d in data[:5]:
    print(f'日期: {d.date}, 费用: {d.cost}, CPC: {d.cpc}, 预算: {d.budget}')
EOF
```
