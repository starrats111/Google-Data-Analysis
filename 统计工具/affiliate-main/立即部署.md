## 🎉 功能说明

### 💰 可提现金额
- **来源**: 数据库查询
- **条件**: Approved + 有 settlement_date + 无 paid_date
- **特点**: 不受日期筛选影响

### ⏳ 提现中金额
- **来源**: withdrawal_requests 表
- **条件**: status='processing'
- **特点**: 目前为 $0（还没有提现请求）

### 💵 已提现佣金
- **来源**: PM Payment Summary API
- **条件**: 根据日期范围
- **特点**: 受日期筛选影响

### 📅 日期筛选
- 最近7天
- 本月（默认）
- 上月
- 最近3个月
- 最近半年
- 全部

**注意**: 只影响"已提现佣金"，不影响"可提现金额"。
