# 正确的启动命令

## 重要提示
**每个命令要单独执行，不要连在一起！**

## 正确的步骤

### 1. 进入目录并激活虚拟环境

```powershell
cd "D:\Google Analysis\backend"
.\venv\Scripts\Activate.ps1
```

### 2. 安装依赖（一个一个执行）

```powershell
pip install fastapi
pip install uvicorn
pip install sqlalchemy
pip install pydantic
pip install pydantic-settings
pip install python-jose[cryptography]
pip install passlib[bcrypt]
pip install python-multipart
pip install openpyxl
pip install python-dotenv
```

或者一次性安装（推荐）：

```powershell
pip install fastapi uvicorn sqlalchemy pydantic pydantic-settings python-jose passlib python-multipart openpyxl python-dotenv
```

### 3. 初始化数据库（每个命令单独执行）

```powershell
python scripts/init_db.py
```

等待完成，然后：

```powershell
python scripts/init_users.py
```

等待完成，然后：

```powershell
python scripts/init_platforms.py
```

### 4. 启动服务

```powershell
uvicorn app.main:app --reload
```

注意：是 `--reload`（两个短横线），不是 `-reload`

## 常见错误

### ❌ 错误示例
```powershell
python scripts/init_platforms.pyuvicorn app.main:app -reload
```
命令被连在一起了！

### ✅ 正确做法
```powershell
python scripts/init_platforms.py
```
按回车执行，等待完成后再执行下一个命令。

## 验证安装

安装完成后，验证关键包：

```powershell
python -c "import fastapi; print('FastAPI OK')"
python -c "import sqlalchemy; print('SQLAlchemy OK')"
python -c "import uvicorn; print('Uvicorn OK')"
```

如果都显示 "OK"，说明安装成功。
