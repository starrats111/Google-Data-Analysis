# Google Ads API Quota Increase Request - Answer Template

## Question: Why do you need a higher new user authorization rate limit?

### Recommended Answer (English):

We are an advertising agency providing Google Ads campaign management services for multiple employees. Our business model requires us to manage Google Ads accounts for multiple employees through MCC (My Client Center) accounts.

**Business Model:**
- We manage Google Ads campaigns for multiple employees via MCC accounts
- We provide advertising services including campaign setup, optimization, and data analysis
- We need to regularly generate advertising performance reports and data analysis for our employees

**How we use Google Ads API:**
- We use the API to automatically synchronize advertising data (campaigns, costs, clicks, impressions, etc.) from multiple employee accounts
- We integrate this data into our data analysis platform to provide real-time reports for employees
- We perform automated data synchronization to reduce manual operations and improve efficiency
- We support batch data synchronization and analysis across multiple accounts

**Current Problem:**
- We are currently using Explorer Access (2,880 calls/day)
- Our daily quota is frequently exhausted before we can complete data synchronization for all employee accounts
- This impacts our business operations and prevents us from providing timely reports to our employees
- We need to upgrade to Standard Access to support our business growth and ensure reliable data synchronization

**Expected Usage:**
- We manage approximately 11 active MCC accounts
- Each MCC account may contain multiple customer accounts
- We need to synchronize data for the past 7 days on a regular basis
- This requires a significant number of API calls to ensure all employee accounts are properly synchronized

**Request:**
We request an increase in the new user authorization rate limit to support our business operations and ensure we can provide reliable data synchronization services to all our employees.

- 配额经常在一天内耗尽，无法完成所有员工账号的数据同步
- 影响业务运营，无法及时为员工提供报告

我们请求增加新用户授权率限制，以支持我们的业务运营，确保能够为所有员工提供可靠的数据同步服务。
