# 简化安装指南 - 快速启动

## 问题
系统缺少 C/C++ 编译器，导致某些包无法编译。

## 快速解决方案

### 步骤1：安装基础依赖（不包含 pandas）

```powershell
cd "D:\Google Analysis\backend"
.\venv\Scripts\Activate.ps1
pip install fastapi uvicorn[standard] sqlalchemy pydantic pydantic-settings python-jose[cryptography] passlib[bcrypt] python-multipart openpyxl python-dotenv
```

### 步骤2：安装 numpy（使用预编译版本）

```powershell
pip install numpy
```

### 步骤3：安装 pandas（使用旧版本，有预编译）

```powershell
pip install pandas==2.0.3
```

### 步骤4：暂时跳过 psycopg2-binary

如果 PostgreSQL 还没安装，可以先跳过：

```powershell
# 暂时不安装 psycopg2-binary
# 稍后安装 PostgreSQL 后再安装
```

### 步骤5：修改配置使用 SQLite（临时方案）

如果暂时没有 PostgreSQL，可以修改配置使用 SQLite：

编辑 `backend/app/config.py`，将：
```python
DATABASE_URL: str = "postgresql://admin:password@localhost:5432/google_analysis"
```

改为：
```python
DATABASE_URL: str = "sqlite:///./google_analysis.db"
```

### 步骤6：初始化数据库

```powershell
python scripts/init_db.py
python scripts/init_users.py
python scripts/init_platforms.py
```

### 步骤7：启动服务

```powershell
uvicorn app.main:app --reload
```

## 验证

访问 http://localhost:8000/docs 应该能看到 API 文档。
