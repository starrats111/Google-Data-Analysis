# 安装缺失的依赖

## 问题
缺少 `zustand` 依赖包，导致前端无法启动。

## 解决方法

在 PowerShell 中，进入 frontend 目录并安装依赖：

```powershell
cd "D:\Google Analysis\frontend"
npm install
```

或者只安装 zustand：

```powershell
cd "D:\Google Analysis\frontend"
npm install zustand
```

## 安装完成后

依赖安装完成后，Vite 会自动重新编译，错误应该会消失。

如果 Vite 服务还在运行，它会自动检测到新安装的依赖并重新加载。

## 如果 Vite 服务已停止

重新启动前端服务：

```powershell
cd "D:\Google Analysis\frontend"
npm run dev
```
