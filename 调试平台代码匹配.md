# 调试平台代码匹配问题

## 问题分析

API URL字段没有显示，可能的原因：

1. **前端代码未更新**：Cloudflare Pages可能还没有重新部署
2. **平台代码不匹配**：数据库中可能是"RW"（大写），需要确保能匹配到"rw"配置
3. **条件渲染逻辑问题**：可能在某些情况下没有正确触发

## 解决方案

### 方案1：检查平台代码

在浏览器控制台中执行：
```javascript
// 查看当前账号的平台信息
console.log('Platform code:', syncAccount?.platform?.platform_code)
console.log('Platform name:', syncAccount?.platform?.platform_name)
```

### 方案2：确保配置支持所有变体

确保`platformApiConfig.js`中同时支持：
- `rw`（小写）
- `RW`（大写）
- `rewardoo`（全名）

### 方案3：添加调试日志

在代码中添加console.log来调试：
```javascript
console.log('Selected platform:', selectedPlatform)
console.log('Platform code:', selectedPlatform?.platform_code)
console.log('Platform config:', platformConfig)
```
