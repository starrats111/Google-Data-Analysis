# 检查前端服务状态

## 问题
浏览器显示 "localhost 拒绝连接"，说明 Vite 开发服务器没有运行。

## 解决方法

### 1. 检查 Vite 服务是否在运行

查看运行 `frontend_start.ps1` 的 PowerShell 窗口，应该看到类似这样的输出：
```
VITE v5.4.21 ready in 751 ms
➜  Local:   http://localhost:3000/
```

如果没有看到这个，说明服务没有启动。

### 2. 重新启动前端服务

在 PowerShell 中执行：

```powershell
cd "D:\Google Analysis\frontend"
npm run dev
```

### 3. 等待服务启动

启动成功后，您会看到：
- "VITE v5.4.21 ready in XXX ms"
- "Local: http://localhost:3000/"

### 4. 然后刷新浏览器

看到服务启动后，刷新浏览器页面（按 F5）。

## 如果服务启动失败

检查是否有错误信息，常见问题：
- 端口 3000 被占用
- 依赖未完全安装
- Node.js 版本问题
